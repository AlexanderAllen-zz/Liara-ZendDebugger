Liara/Zend Debugger
=============

Ansible role for installing Zend Debugger.

Requirements
------------

- PHP
- AlexanderAllen.Liara.

Role Variables
--------------

The variables used by this role are provided by `AlexanderAllen.Liara`.

Dependencies
------------

 - `AlexanderAllen.Liara`

Example Playbook
----------------

    - hosts: webservers
      roles:
         - { role: AlexanderAllen.ZendDebugger }
